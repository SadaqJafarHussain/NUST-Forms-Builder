## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/en) (Version: >=18.x)
- [pnpm](https://pnpm.io/)
- [Docker](https://www.docker.com/) - for PostgreSQL

### Installation

1. Clone the repository
   ```sh
   git clone https://github.com/SadaqJafarHussain/NUST-Forms-Builder.git
   ```

2. Install dependencies
   ```sh
   pnpm install
   ```

3. Set up environment variables
   ```sh
   cp .env.example .env
   ```

4. Start the development server
   ```sh
   pnpm dev
   ```

## Usage

1. Access the application at `http://localhost:3000`
2. Create an account or sign in
3. Start creating surveys using the survey editor
4. Share surveys via links and collect responses
5. Analyze responses in the dashboard
